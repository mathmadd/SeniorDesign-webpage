About The Project

Stock Price Prediction Using FPGA

Introduction:

Stock prediction is a very difficult task as there are several factors that affect the prices of stocks. Many researchers suggest that stock prices can be predicted with some important indicators (RSI, MACD, and others). Even though we predict the prices, these prices are only valid for a small-time frame. We need fast computers or computing units that can predict these prices faster and help the user to place orders at a faster rate.

Problem Description:

The goal of this project is to design a stock price prediction system using a smart algorithm and implement the algorithm using an FPGA. Since it will be implemented on an FPGA, it is expected to run faster and help users to place orders faster. To understand the working of the prediction algorithm, the algorithm can be implemented on a software platform and then compared with the hardware (FPGA) implementation.

High Level Requirements:

  1. Real time prediction of the stock prices (Day’s high and Day’s low) using RSI, Bollinger’s band, Stochastic Oscillator, and MACD.
  2. Implement the prediction algorithm on an FPGA hardware and on a software platform.
  3. Compare the performance of software and hardware algorithms.
  4. Use a web server to feed the stock details to the software and the hardware systems and display the prediction results.
